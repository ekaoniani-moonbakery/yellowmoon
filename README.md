# Yellow Moon Bakery

A modern, visually stunning bakery landing page featuring WebGL animated backgrounds and elegant design.

## Features

- **WebGL Animated Background**: Procedurally generated moon and cloud animations
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean, minimalist interface with smooth animations
- **Multi-Page Navigation**: Home, Menu, Story, and Visit pages

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- WebGL for graphics rendering

## Installation

1. Clone this repository:
```bash
git clone https://github.com/yourusername/yellow-moon-bakery.git
```

2. Open `index.html` in your web browser

That's it! No build process or dependencies required.

## File Structure

```
yellow-moon-bakery/
├── index.html          # Main HTML file
├── script.js           # JavaScript and WebGL logic
└── README.md           # This file
```

## Customization

### Colors
The color palette can be modified in the CSS variables and WebGL shader code:
- Primary: #f2c55c (Yellow)
- Secondary: #e8d4a8 (Cream)
- Background: Dark gradients

### Content
Edit the HTML file to update:
- Menu items
- Business hours
- Contact information
- About text

## Browser Compatibility

- Chrome (recommended)
- Firefox
- Safari
- Edge

WebGL is required for the animated background. If WebGL is not supported, a fallback gradient background is displayed.
